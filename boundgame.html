Bound Game
"Bound Game", the project title for a game that lacks a proper title, is an Earthbound inspired game I am developing in my spare time.



Development began in September 2016 and is planned to continue into 2018.

I am however considering a Halloween 2017 release.


Planned platforms for release are Windows, Mac, Linux, and possibly some modern consoles.


The game is being developed with the following tools:
  • C# Language
  • Microsoft Visual Studio Community
  • MonoGame Framework
  • MonoGame Extended Framework
  • Tiled Map Editor
  • GIMP
  • Audacity


So far, the game has a fair amount of features that are complete: character, party, leveling, items, npc scripting, and equipping and ability systems.


The main characters in the game are largely based off people in my real life. The setting has a sense of modern societal humor, but ventures into deeper and darker undertones such as consipiracy theories and the supernatural.


The foundation is built, now to draw hundreds of sprites, tiles, maps, and dialogue!
